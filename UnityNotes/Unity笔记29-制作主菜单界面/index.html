Unity笔记29-制作主菜单界面

Created2021-08-05|Updated2021-08-05|Unity学习笔记
|Post View:|Comments:

本章我们来为游戏制作一个主菜单界面

在新场景中绘制画布

我们要专门为我们的菜单新建一个场景，而菜单则是在这个新场景中绘制的

我这里先把场景搭建好了，注意我们在摆放人物的时候可以点击右键菜单进行脱包处理，这样我们移除组件时就不会影响到素材里面的模板了

然后我们也把UI摆放好

同时我们也可以给UI添加一些视觉效果的处理

将UI添加立体效果

这里可以添加一种效果，我想点击new game 时候相机会穿过标题UI然后靠近传送门的样式，下面就来实现一下

我们选中画布，然后如下图设置，将Render Mode选择为Camera，再把摄像机拖到下面的框框里

这时候点击运行会发现按钮给场景遮挡住了，这时候我们可以调一下Plane Distance，比如改成1，按钮就又可以看到了

最后我们再把Render Mode改为World Space，然后尝试向前移动相机，就可以发现达成了相机穿越标题的效果了，之所以要经过上面的步骤是因为可以让Unity自动调节UI的合适位置

编写代码

UI建立好之后我们就要来实现按钮的功能了

按钮最主要就是监听事件，下面先单独来写各种功能的方法，最后再配上整个代码文件

Exit

其中最简单的就是退出游戏，一行搞定

void QuitGame(){//退出游戏功能
        Application.Quit();
    }

NewGame

然后是创建新游戏，这里就要涉及多个代码文件

//MainMenu.cs
void NewGame(){
        //清楚所有存档
        PlayerPrefs.DeleteAll();

        //转换场景
        //会在SceneController里面完成
        SceneController.Instance.TransitionToFirstLevel();//加载第一个场景
    }
//SceneController.cs

//转换场景，可用于在标题界面过渡到第一个界面中
    IEnumerator LoadLevel(string scene){
        if(scene!=""){//传入参数不为空
            //加载场景
            yield return SceneManager.LoadSceneAsync(scene);
            //加载人物，在GMaeManager中创建了一个用于获得初始位置的方法
            yield return player = Instantiate(playerPrefab,
            GameManager.Instance.GetEntrance().position,
            GameManager.Instance.GetEntrance().rotation);

            //保存数据
            SaveManager.Instance.SavePlayerData();
            yield break;//结束协程
        }
    }

    //加载第一个场景，可以将这个直接放在标题界面的功能里
    public void TransitionToFirstLevel(){
        StartCoroutine(LoadLevel("3D RPG"));
    }
//GmaeManager.cs
//获得场景加载入口，这里是标题界面进入主场景时用于获得落地点
    public Transform GetEntrance(){
        foreach(var item in FindObjectsOfType<TransitionDestination>()){
            if(item.destinationTag==TransitionDestination.DestinationTag.ENTER)
                return item.transform;
        }
        return null;
    }

注意要把几个Manager对象复制到标题场景中

Continue

由于我们游戏有多个场景，因此在加载功能中我们还要记住上次保存玩家是停留在哪一个场景，从而在加载存档时判断要加载哪一个场景的画面

也是有多个代码文件需要修改，需要注意区分

//SaveManager.cs

	string sceneName="";//记录保存场景的名字
    public string SceneName{get{return PlayerPrefs.GetString(sceneName);}}

    void Save(UnityEngine.Object data,string key){
        var jsonData = JsonUtility.ToJson(data);//转换为json
        PlayerPrefs.SetString(key,jsonData);//再建立键值保存

        //在保存数据同时保存玩家停留在哪个场景
        PlayerPrefs.SetString(sceneName,SceneManager.GetActiveScene().name);

        PlayerPrefs.Save();
    }
//SceneController.cs
	//Continue
    //用于加载游戏的加载方法
    public void TransitionToLoadGame(){
        string sceneName = SaveManager.Instance.SceneName;//获得上次保存场景名称，可能为空
        if(sceneName!=""){
            //不为空的话就加载场景
            StartCoroutine(LoadLevel(sceneName));
        }
    }
//PlayerController.cs
void OnEnable(){
		//素材启用时注册事件
		MouseManager.Instance.OnMouseClicked+=MoveToTarget;
		MouseManager.Instance.OnEnemyClicked+=EventAttact;
		GameManager.Instance.RigisterPlayer(charcterStates);//注册GmaeManager
	}

	void Start(){
		//加载人物数据
		SaveManager.Instance.LoadPlayerData();
	}
//MainMenu.cs
    void ContinueGame(){
        //转换场景，读取进度
        SceneController.Instance.TransitionToLoadGame();
    }

游戏中Esc返回主菜单

我们还有一个功能是在游戏内必要的，就是返回主菜单功能，这里也来实现一下

//SaveManager.cs
void Update(){
       if(Input.GetKeyDown(KeyCode.Escape)){
           //游戏中Esc回到主菜单
           SceneController.Instance.TransitionToMain();
       }

       if(Input.GetKey(KeyCode.S))//保存
           SavePlayerData();

       if(Input.GetKey(KeyCode.L))//读取
           LoadPlayerData();
   }
//SceneController.cs
//加载标题场景协程
    IEnumerator LoadMain(){
        yield return SceneManager.LoadSceneAsync("Main");
        yield break;
    }

    //加载标题方法
    public void TransitionToMain(){
        StartCoroutine(LoadMain());
    }

到这我们就成功实现了主菜单到游戏再到主菜单的过渡效果
