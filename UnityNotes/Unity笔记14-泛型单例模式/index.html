Unity笔记14-泛型单例模式

Created2021-07-20|Updated2021-07-21|Unity学习笔记
|Post View:|Comments:

这一章我们来编写一个泛型单例模式的代码，这是因为我们的项目中会有很多的Manager，如果每个都使用单例模式会更好地去管理和使用

例如其中控制我们游戏的开始暂停等一般是用Game Manager来命名，我们来创建一个代码文件以及一个空对象使得代码可以依附在它上面

这个代码我们先放在一边之后再补充，先把泛型单例模式代码写了先

泛型单例模式

创建目录文件如下

public class Sigltion<T> : MonoBehaviour where T : Sigltion<T>{

    private static T instance;

    public static T Instance{
        get{
            return instance;
        }
    }

    //允许子类对Awake进行重写
    protected virtual void Awake(){
        if(instance != null)
            Destroy(this.gameObject);
        else
            instance = (T)this;
    }

    //检测单例是否生成
    public static bool IsInitialized{
        get{return instance != null;}
    }

    //允许子类修改销毁时的操作
    protected virtual void OnDestroy(){
        if(instance == this)
            instance = null;
    }

}

这样一旦一个A类继承了Sigltion<A>之后，我们就可以在其它类中调用A类的单例，如：A.Instance
