Unity笔记12-实现攻击数值计算

Created2021-07-18|Updated2021-07-26|Unity学习笔记
|Post View:|Comments:

这一章节中我们来实现攻击伤害的计算，给予人物或敌人真实的攻击伤害

补充人物攻击动画

在之前我们要给人物添加一下暴击动画，和上一章节设置敌人动画类似，注意要设置动画切换效果

打开PlayerController.cs，修改一下

IEnumerator MoveToAttackTarget(){
    agent.isStopped = false;
    transform.LookAt(attactTarget.transform);

    while(Vector3.Distance(transform.position,attactTarget.transform.position)>
          charcterStates.attackData.attackRange){
        //比较人物和敌人的距离是否大于0攻击距离，Distance()可以比较两个Vector3距离
        this.agent.destination = attactTarget.transform.position;

        yield return null;
    }

    agent.isStopped = true;

    //攻击动画和CD都在下面
    if(lastAttactTime<0){//CD冷却完成可执行攻击
        animator.SetBool("Critical",charcterStates.isCritical);//增加了传递是否暴击
        animator.SetTrigger("Attack");//普通攻击
        lastAttactTime = charcterStates.attackData.coolDown;//修改了这里的CD时间
    }
}

这里我们修改了两个地方，一个是增加了暴击判断，另一个是修改了CD时间判断

增加攻击数值计算

前面补充完成后，我们就可以正式来写攻击伤害的数值计算了

打开CharcterStates.cs，我们在最下面加上我们的伤害计算代码

void EventAttact(GameObject target){
    if(target!=null){//如果鼠标点击的目标不为空，或者点击的敌人不是属于死亡状态
        attactTarget = target;
        //随机暴击率语句
        charcterStates.isCritical = Random.value < charcterStates.attackData.criticalChance;
        StartCoroutine(MoveToAttackTarget());
    }
}

private int CurrentDamage(){
    //在攻击力范围内取随机数返回一个随机攻击力
    float coreDamage = Random.Range(attackData.minDamage,attackData.maxDamage);

    if(isCritical){//如果暴击的话还要计算上暴击率
        coreDamage *=attackData.criticalMultiplier;//乘等于
        Debug.LogFormat("暴击了！伤害为（不算目标防御力）：{0}",coreDamage);
    }

    return ((int)coreDamage);
}

#endregion

调用伤害计算方法

伤害计算代码写好后，我们就要去想如何调用它，在哪里调用它了

参考一些RPG游戏，一般都是当你的武器挥出去后接触到敌人那一瞬间计算伤害，我们也可以按照这样来做

在人物身上调用

首先要调出Animation窗口

然后窗口左上角选择上我们的攻击动画Attack01，我们就可以在攻击动画里面的某个特定帧添加事件了

我们再次编辑一下PlayerController.cs代码加一个用来调用的方法

//Animation Event
void Hit(){
    var targetStates = attactTarget.GetComponent<CharcterStates>();

    //计算伤害
    targetStates.TakeDamage(charcterStates,targetStates);

}
