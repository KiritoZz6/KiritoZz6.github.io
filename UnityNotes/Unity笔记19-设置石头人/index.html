Unity笔记19-设置石头人

Created2021-07-25|Updated2021-07-25|Unity学习笔记
|Post View:|Comments:

这一章节我们来设置石头人，由于和设置兽人相差不大，篇幅比较少

设置动画

我们去浏览石头人的动画可以发现，Attack01是类似拿起石头丢我们，它才是技能攻击，而Attack02是击晕玩家的攻击动画，可以作为普通攻击来使，因此我们在调整攻击动画时要注意分清

我们的动画控制器文件就选择是基于兽人的动画来完成，创建一个New Animator Override Controller，模板选择为兽人，然后再一个个动画替换上去

其中Run并没有自带动画，我们可以用Walk代替

编写代码文件

我们像兽人代码文件那样单独创一个代码文件给石头人用，并且也是要继承EnemyController

public class Golem : EnemyController{

    [Header("Skill（石头人特殊攻击设置）")]
    public float kickForce = 20;//推开敌人距离

    void kickOff(){
        //石头人击飞玩家
        if(attackTarget != null && transform.IsFacingTarget(attackTarget.transform)){
            var targetState = attackTarget.GetComponent<CharacterStates>();

            transform.LookAt(attackTarget.transform);
            Vector3 direction = (attackTarget.transform.position - transform.position).normalized;

            attackTarget.GetComponent<NavMeshAgent>().isStopped = true;
            attackTarget.GetComponent<NavMeshAgent>().velocity = direction*kickForce;
            targetState.TakeDamage(this.characterStates,targetState);
        }
    }

}

最后再添加动画事件就行了
