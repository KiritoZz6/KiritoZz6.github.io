Unity笔记21-反击石头人

Created2021-07-27|Updated2021-07-28|Unity学习笔记
|Post View:|Comments:

这一章中我们来实现石头人丢出石头来击飞玩家并且造成伤害的效果，同时玩家还可以利用石头反击石头人

实现石头击飞玩家并造成伤害

我们打开Rock.cs，我们要有区分性的触发伤害，对于玩家和敌人可以造成伤害，但是对于丢到地面的情况是不能造成伤害的，所以这里可以利用一个枚举来区分开来

public enum RockStates{HitPlayer,HitEnemy,HitNothing};
public  RockStates rockStates;//枚举实例

public int damage;//造成的伤害是多少


void OnCollisionEnter(Collision other){//other指被石头打到的对象
    //当发生碰撞时会执行这个函数
    switch(rockStates){
        case RockStates.HitPlayer:
            if(other.gameObject.CompareTag("Player")){
                other.gameObject.GetComponent<NavMeshAgent>().isStopped = true;
                other.gameObject.GetComponent<NavMeshAgent>().velocity = direction * force;//击退玩家

                other.gameObject.GetComponent<Animator>().SetTrigger("Dizzy");//眩晕

                //这里的TakeDamage是重载的其中一个方法
                //造成伤害
                other.gameObject.GetComponent<CharacterStates>()
                    .TakeDamage(damage,other.gameObject.GetComponent<CharacterStates>());

                rockStates = RockStates.HitNothing;//复原状态以防一直执行
            }
            break;
    }//end switch
}

我们看到其中的TakeDamage()是一个重载方法，打开CharacterStates.cs我们这里补充一下

//函数重载
public void TakeDamage(int damage,CharacterStates defener){
    int dam = Mathf.Max(damage - defener.currentDefence,0);
    defener.currentHealth = Mathf.Max(defener.currentHealth - dam,0);
}

然后保存关闭，别忘了要在外边石头对象上赋予一个初始伤害值

这样我们就实现了石头的第一个状态

实现石头攻击石头人

先把代码完善好

case RockStates.HitEnemy:
if(other.gameObject.GetComponent<Golem>()){
    //获得了石头人代码组件的就是石头人的，否则返回的是空
    var otherStates = other.gameObject.GetComponent<CharacterStates>();
    otherStates.TakeDamage(damage,otherStates);

    Destroy(gameObject);
}
break;

因为我们这里是反击石头人，所以要有个判断鼠标点击到了石头，一个最直接的办法就是给我们的石头添加一个标签，然后再修改一下鼠标点击时的方法

回到PlayerController.cs修改一下

void Hit(){
    if(attactTarget.CompareTag("Attackable")){
        //如果点击的对象不是敌人而是其它另类可攻击对象，如石头
        if(attactTarget.GetComponent<Rock>() && attactTarget.GetComponent<Rock>().rockStates 
           == Rock.RockStates.HitNothing){
            //如果是石头并且石头不在空中而是落地之后
            attactTarget.GetComponent<Rock>().rockStates = Rock.RockStates.HitEnemy;

            //防止攻击那一刹那速度小于1导致石头状态发生改变
            attactTarget.GetComponent<Rigidbody>().velocity = Vector3.one;
            
            //添加一个玩家朝向的力，力的大小暂时定 40
            attactTarget.GetComponent<Rigidbody>()
                .AddForce(transform.forward * 40,ForceMode.Impulse);
        }

    }else{
        var targetStates = attactTarget.GetComponent<CharacterStates>();
        //计算伤害
        targetStates.TakeDamage(charcterStates,targetStates);
    }
}

我们可以看到条件中有一个是判断是否为HitNothing，我们可以利用判断石头是否为接近于静止时为条件，这里我们就要回去Rock.cs做一些修改了

void Start(){
    rb = GetComponent<Rigidbody>();
    rb.velocity = Vector3.one;//防止石头一生成时速度为0导致状态发生改变
    FlyToTarget();
}

void FixedUpdate(){
    //固定帧更新
    if(rb.velocity.sqrMagnitude < 1f){//向量的平方
        rockStates = RockStates.HitNothing;
    }
}

为石头加上爆裂特效

最后我们想在石头打到石头人时有一个爆裂碎石特效，这里就要用到Particle System了

创建好之后我们按照下图参考设置

自己多多测试一下效果，满意之后我们就可以改个名将它保存为一个素材放好先，然后不要忘了把地图上刚刚调试的删除

public GameObject RockBreakParticle;//石头爆裂特效，供外界拖拽进来

case RockStates.HitEnemy:
if(other.gameObject.GetComponent<Golem>()){
    //获得了石头人代码组件的就是石头人的，否则返回的是空
    var otherStates = other.gameObject.GetComponent<CharacterStates>();
    otherStates.TakeDamage(damage,otherStates);

    Destroy(gameObject);
